# jm-mobile

这是一个禁漫的第三方安卓客户端，基于 Android Compose 构建。

## NSFW 警告⚠

本软件可能存在裸露、暴力、色情或冒犯等不适宜公众场合的内容，请勿在公共场合使用本软件，避免不必要的纷争。

## 功能

- [x] 登录
- [x] 搜索
- [x] 阅读
- [x] 收藏
- [x] 简易用户信息
- [x] 阅读历史
- [x] 漫画详情
- [x] 历史评论
- [x] 漫画评论
- [x] 发表评论
- [x] 首页推荐
- [x] 目录阅读
- [x] 日夜间模式
- [x] 每周必看
- [x] 自动登录
- [ ] 相关漫画
- [ ] 漫画下载
- [ ] 漫画分类搜索
- [ ] 图片分流
